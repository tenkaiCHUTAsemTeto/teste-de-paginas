# Landing Page - Novo Produto

Este é um template de landing page para anúncio de produto, pronto para hospedagem no GitHub Pages.

## Como Usar

1. **Crie um novo repositório no GitHub**
   - Vá para github.com/new
   - Nomeie como `meu-produto` (ou outro nome)
   - selecione "Public"

2. **Envie os arquivos**
   - Faça upload dos arquivos `index.html` e `style.css`

3. **Ative o GitHub Pages**
   - Vá em Settings > Pages
   - Em "Branch", selecione `main` (ou `master`)
   - Clique em Save

4. **Seu site estará online**
   - A URL será: `seu-usuario.github.io/nome-do-repositorio`

## Como Personalizar

### Alterarcores
Edite o arquivo `style.css` e modifique os valores em `:root`:
```css
:root {
    --primary: #ff4500;      /* Cor principal */
    --secondary: #1a1a1a;   /* Cor secundária */
}
